Política de Privacidade

Última atualização: 09/06/2026

O DespensAI ("aplicativo", "nós") é um produto da Nexus Orbital Inova Simples (CNPJ 66.465.235/0001-67), Santo André/SP. Esta política explica quais dados o aplicativo coleta e como são usados. Ao usar o DespensAI, você concorda com os termos abaixo.

1. Quais dados coletamos

  • Dados que você cadastra: itens dos seus estoques (casa, pet, farmácia etc.), registros de dieta, receitas e listas de compra.
  • Fotos: quando você fotografa uma nota fiscal, um produto ou um prato, a imagem é processada para identificar os itens. As fotos são usadas apenas para esse fim.
  • Localização (opcional): se você permitir, usamos sua localização aproximada para mostrar mercados e lojas próximas. Você pode recusar.
  • Dados de uso: informações básicas de funcionamento do app para corrigir erros e melhorar a experiência.

2. Como usamos seus dados

  • Para organizar seus estoques, dieta e compras dentro do app.
  • Para recomendar produtos e montar seu carrinho de compras.
  • Para melhorar o aplicativo.

Não vendemos seus dados pessoais.

3. Links de afiliado

O DespensAI participa de programas de afiliados (como o Programa de Associados da Amazon). Quando você compra por um link recomendado no app, podemos receber uma comissão, sem custo adicional para você. A decisão de compra é sempre sua.

4. Compartilhamento

Não compartilhamos seus dados pessoais com terceiros, exceto serviços necessários para o funcionamento do app (por exemplo, processamento de imagem) e quando exigido por lei.

5. Seus direitos (LGPD)

Você pode solicitar acesso, correção ou exclusão dos seus dados a qualquer momento pelo e-mail abaixo. Você também pode excluir seus dados apagando o aplicativo do seu dispositivo.

6. Crianças

O DespensAI não é direcionado a menores de 13 anos.

7. Contato

Dúvidas sobre privacidade: contato@nexusorbital.com.br
